BtoCマッチングサイト、システム

家を建てたいユーザーと建築会社をマッチングさせるためのシステムです。
ユーザーがフォームから申し込みを行い管理システムにデータが入ります。その内容を元に建築会社用フォームを生成します。建築会社からの回答を集め、ユーザー用フォームを生成します。ユーザーがマッチング希望会社を選択しマッチング完了です。

設計、フロントエンド、バックエンド

ユーザーからの申込み内容を表示、マッチングの進行の確認等を行う機能を既存システムへ追加。
PHP, Codeigniter, MySQL, HTML

建築会社がユーザーの申込み内容を確認しマッチングの希望を回答するフォーム。
URL生成、メール、リマインドメール、フォーム生成、フォーム送信内容検証、回答管理、締め切りなどの機能を実装。
PHP, MySQL, HTML, css(bootstrap), javascript

ユーザーが建築会社を選択してマッチングを完了させるフォーム。
URL生成、メール、リマインドメール、フォーム生成、フォーム送信内容検証、回答管理などの機能を実装。
PHP, MySQL, HTML, css(bootstrap), javascript
